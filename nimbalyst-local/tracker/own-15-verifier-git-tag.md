Публичный верификатор перепинен на реальный SHA `f560aed8` (manifest v1.2). Скептик может скачать
и проверить нас. Сейчас download-ссылка указывает на `main` (работает, но меняется во времени).
Git-тег даст «золотую» неизменяемую ссылку — сильнее для доверия. Я умею пушить только файлы,
не git-теги, поэтому нужен ты.

