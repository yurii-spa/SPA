# inbox/ — брось сюда заметку, и она станет заданием

Это папка быстрого захвата заданий из **Obsidian** (хранилище = корень репо).

**Как пользоваться:** создай здесь markdown-файл с парой строк — что нужно сделать. Первая строка
станет названием задания. Больше ничего не требуется.

Пример `inbox/почини-график.md`:
```
Почини график на дашборде — на телефоне вылезает за экран
Проверь и десктоп заодно
```

**Что дальше:** оркестратор в начале цикла выполняет
`python3 scripts/orchestrator_queue.py ingest-notes` — каждая заметка превращается в карточку
`inbox` в трекере (source: obsidian), а оригинал уезжает в `inbox/.ingested/`. Затем карточка
классифицируется (задача / идея / непонятно) по протоколу `docs/ORCHESTRATOR_PROTOCOL.md`.

Другие входы для заданий (всё ведёт в одни файлы-карточки): карточка в Nimbalyst или Telegram-бот
(`/task …` либо голосовое сообщение — расшифрую офлайн).

_README и `.ingested/` при разборе пропускаются._
