## What it is

SPA — автономная система управления stablecoin-портфелем в DeFi. LLM-агенты
(CEO, Alpha, Research, Reporting) непрерывно анализируют рынок, ищут источники
доходности и готовят предложения — но **ни одна строка кода, касающаяся
капитала, не содержит LLM**. Аллокация, риск-лимиты и исполнение — строго
детерминированное ядро с версионируемой Risk Policy, изменяемой только через
ADR-процесс (по сути — инвестиционный комитет в коде). Каждое решение
логируется в audit trail и воспроизводимо: snapshot данных → предложение →
риск-вердикт → сделка.

Архитектурный тезис: институциональный капитал не придёт в «AI-трейдинг» из-за
prompt injection и галлюцинаций — и правильно сделает. SPA устраняет сам класс
этих рисков границей «deterministic core / LLM shell», оформленной кодом и
CI-гейтами, а не обещаниями. SPA не держит собственные стратегии в смарт-
контрактах — она маршрутизирует капитал между чужими аудированными протоколами
(Aave, Morpho, Pendle и далее 20+), что минимизирует собственную contract-
поверхность и позволяет расширять покрытие со скоростью кода, а не governance.

## Why this approach is defensible

1. **Deterministic core / LLM shell.** Капитала касается только детерминированный
   код; LLM-агенты предлагают, ядро валидирует и исполняет. CI запрещает import
   LLM-клиентов в risk/execution-слои. Ни один крупный агрегатор так не позиционируется.
2. **Audit trail каждого решения.** Полная цепочка «данные → предложение →
   риск-вердикт → сделка» с correlation id. Phase 4: Merkle-root лога публикуется
   on-chain ежедневно — **трек физически нельзя переписать задним числом**
   (Proof-of-Track, верифицируется третьей стороной).
3. **Risk-first как продукт.** Версионируемая Risk Policy: лимиты по осям риска
   (credit / peg / duration / bridge), exit-latency бюджет (доля позиций с выходом
   >72ч ≤ 25%), kill-switch с drill-замерами, стресс-сценарии (LUNA 2022,
   USDC depeg 2023) еженедельно. Изменения политики — только через ADR + paper-тест.
4. **Агрегатор агрегаторов.** Меньше собственного contract-риска, быстрее
   масштабирование покрытия, B2B-монетизация мозга (white-label API) без кастоди.

## Roadmap to $100M AUM

| Веха | Срок | Критерий прохождения |
|---|---|---|
| 30+ дней верифицируемого paper-трека | 2026-07 | ни одного gap'а, Sharpe/Sortino на n≥30 |
| APY ≥5.5% устойчиво, 20+ протоколов, 3 сети | 2026-09 | 60+ дней трека, capacity-adjusted модель |
| Полный агентный слой (автономия 95%) | 2026-10 | неделя работы без ручного вмешательства |
| Live-пилот собственным капиталом ($10–50K → $250K–1M) | 2026-Q4 | Gnosis Safe + Zodiac Roles, mainnet-fork E2E, 0 инцидентов |
| ERC-4626 vault + аудит tier-1 фирмы | 2026-Q4 → 2027-Q1 | аудит без открытых критических находок |
| Первый внешний AUM ($1–3M: DAO-казначейства) | 2027-H1 | юр. структура, AML-скрининг, Proof-of-Track on-chain |
| $10–20M AUM, white-label API клиенты | 2027-H2 | 12 мес трека |
| $50–100M AUM, ARR $1.5–3M | 2028 | 3 vault-профиля, аудиты #2–3, команда 5+ |

Капитал следует за треком, не наоборот: лимиты AUM повышаются по формальной
лестнице (Capital Ladder) с автоматическим понижением при любом инциденте ≥1% AUM.
Экономика на $50M AUM: 1.5% mgmt + 15% perf + white-label ≈ $1.3–1.6M ARR.
Честный горизонт оценки $100M — 30–36 месяцев; быстрее обещают только те, кто
закладывает токен-спекуляцию вместо выручки.
