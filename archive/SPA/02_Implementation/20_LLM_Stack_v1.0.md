## 1. Принцип распределения моделей

Разные агенты требуют разных профилей: скорость, стоимость, качество рассуждений. Использовать одну модель для всех — неэффективно. Risk Agent и Execution Agent — детерминированный код (не LLM).

---

## 2. Распределение моделей по агентам

| Агент | Реализация | Модель | Обоснование |
|---|---|---|---|
| CEO Agent | LLM | **Claude Sonnet 4.6** | Лучший reasoning для сложных DeFi-решений, надёжный контекст 1M tokens |
| Data Agent | Детерминированный + LLM для аномалий | **Gemini 2.5 Flash-Lite** | Дешёвый и быстрый, используется только для классификации аномалий |
| Risk Agent | **Детерминированный код** | — | LLM запрещён: prompt injection = критический риск |
| Strategy Agent | LLM | **Gemini 2.5 Flash** | Хороший reasoning, hybrid thinking, 1M контекст, цена умеренная |
| Execution Agent | **Детерминированный код** | — | LLM запрещён: не должен иметь доступа к транзакциям |
| Monitoring Agent | Детерминированный + LLM для классификации | **Claude Haiku 4.5** | Быстрый, дешёвый, надёжный для классификации инцидентов |
| Memory & Knowledge Agent | LLM | **Claude Haiku 4.5** | Retrieval и контекст, не критичен по latency |

## 4. Расчёт стоимости

**Параметры:** 5 LLM-агентов × 10 вызовов/день × 2000 tokens input + 500 tokens output

| Конфигурация | В день | В месяц | В год |
|---|---|---|---|
| Текущий стек (гибридный) | ~$0.22 | **~$6.60** | ~$80 |
| Всё на Claude Sonnet | ~$1.13 | ~$33.75 | ~$411 |
| Всё на Gemini Flash-Lite | ~$0.02 | ~$0.60 | ~$7 |

**Текущий стек ($6.60/мес) — ничтожная статья расходов** по сравнению с operational costs провайдеров ($110–125/мес, ADR-009).

## 5. Опциональный локальный запуск

Для изолированного тестирования или снижения costs при масштабировании:

| Модель | VRAM | Скорость | Качество |
|---|---|---|---|
| Gemma 4 26B (MoE) | ~14GB | ~85 tok/s | Очень хорошее |
| Qwen3 32B | 24–48GB | ~60 tok/s | Очень хорошее |

Локальный запуск через **Ollama**. Оправдан при нагрузке >$200/мес (не актуально на текущем этапе).

## 6. Важные ограничения

- **Gemini 2.0 Flash** — deprecated, отключается 01.06.2026. Не использовать.
- Если установлена переменная `ANTHROPIC_API_KEY` в окружении Claude Code — billing идёт по API-тарифам, не по подписке.
- Все LLM-вызовы логируются (model, tokens used, latency) для контроля расходов.
