Правило «всегда бери следующий спринт» имеет смысл, пока есть осмысленная работа. Сейчас её нет: остался либо новый слой косметики, либо задачи, которые физически нельзя закрыть без пользователя. Стэмповать v3.68 значило бы:

Для автономного запуска без присутствия пользователя корректный исход — отчёт, а не write-действие с сомнительной пользой.
