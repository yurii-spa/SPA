### Batch 12 (suggested v10.93): analytics/ automated batch — 158 files
Use sed one-liner (all files have identical pattern):

```bash
# Replace local def + calls in one pass
for f in $(grep -rln "_atomic_write" spa_core/analytics/); do
  # Add import if missing
  grep -q "from spa_core.utils.atomic import" "$f" || \
    sed -i 's/^import json$/import json\nfrom spa_core.utils.atomic import atomic_save/' "$f"
  # Replace calls: _atomic_write_json(data, path) → atomic_save(data, str(path))
  sed -i 's/_atomic_write_json(\(.*\), \(.*\))/atomic_save(\1, str(\2))/g' "$f"
done
```

> **Caution:** Run tests after each file. Some `analytics/` files have subtle variations in argument order.
