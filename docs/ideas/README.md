# docs/ideas/ — свободные идеи владельца

**Агенты по этим файлам НЕ действуют.** Идея ≠ инструкция.

Сюда владелец (через Obsidian или напрямую) кладёт заметки, мысли, наброски. Оркестратор их
**не исполняет** до промоушена: пометка `#promote` в заметке (или явная карточка-задание) →
оркестратор превращает идею в правило (`.claude/rules/` / `CLAUDE.md`), ADR или задачу и
помечает исходник обработанным. Черновики правил — в `docs/rules-draft/`.
