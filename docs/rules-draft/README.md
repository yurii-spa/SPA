# docs/rules-draft/ — черновики правил

**Не действуют, пока не промоутнуты.** Действующие правила живут только в `CLAUDE.md` и
`.claude/rules/`.

Черновик становится правилом после промоушена владельцем (`#promote` или явная карточка):
оркестратор переносит его в `.claude/rules/` или `CLAUDE.md` (при необходимости — ADR) и
отмечает черновик обработанным.
