**Симптом:** коммиты `0586adee` + `173c702c` (мои 2 moat-surface'а) на origin/main HEAD >1 часа, локальные `npm run build` зелёные, но **earn-defi.com их не показывает** — последний задеплоенный коммит `88a081bc`. Сайт НЕ заморожен (88a081bc live, HTTP 200), но CF встал на новейших.

**Влияние:** ВЫСОКОЕ для landing — пока CF не деплоит, НИ ОДИН новый landing-пуш не появится на сайте (мои surface'а + любые будущие). Я приостановил landing-пуши до разблокировки. API/checkup/CI не затронуты.
