# SPA Telegram Bot — Инструкция простыми словами

**Бот называется:** SPA Monitor  
**Команды:** `/status`, `/portfolio`, `/today`, `/week`, `/agents`, `/alerts`, `/pause`, `/resume`, `/help`

---

## Что делает бот?

Бот — это твоя "приборная панель в кармане". Он сидит на твоём Mac 24/7 и следит за тем, что происходит с деньгами ($100K USDC в бумажной торговле). Ты пишешь ему команду — он отвечает. Плюс сам присылает важные уведомления когда что-то происходит.

---

## Команды

| Команда | Что показывает |
|---------|---------------|
| `/status` | Общий статус системы — работает ли всё, сколько денег, какой доход |
| `/portfolio` | Текущее распределение — где лежат деньги, какие протоколы активны |
| `/today` | P&L за сегодня — заработали или потеряли за день |
| `/week` | Недельный отчёт — статистика за 7 дней |
| `/agents` | Список всех 19 агентов с ✅/❌/⏸ статусами |
| `/alerts` | Последние алерты — риски, события, важные новости |
| `/pause` | Остановить торговлю (kill-switch) |
| `/resume` | Возобновить торговлю |
| `/help` | Этот список команд |

---

## Автоматические уведомления

Бот сам пишет тебе (без твоей команды):

| Тип | Когда приходит | Пример |
|-----|---------------|--------|
| 📊 Daily Report | Каждый день в 08:00 | Equity, P&L, APY за день |
| 📅 Weekly Report | Каждый понедельник | Итоги недели |
| 🔴 Red Flag | Немедленно при риске | De-peg стейблкоина, крупный взлом |
| 🗳️ Governance Alert | При важном голосовании | DAO предложение изменить параметры |
| 🎯 Milestone Alert | При достижении цели | Достигли 5% confidence уровня |

**Важно:** каждый тип сообщения отправляется максимум 1 раз в сутки (защита от спама).

## 19 Агентов — кто что делает

Агенты — это отдельные программы, которые работают в фоне на Mac. Каждый запускается автоматически по расписанию.

### 🟢 Торговые агенты (самые важные)

**`daily_cycle`** — Главный торговый движок
- Запускается каждые 30 минут
- Проверяет APY по всем протоколам, принимает решение о размещении средств
- Записывает результат в историю
- *Если ❌ — деньги не работают*

**`portfolio_monitor`** — Следит за портфелем
- Каждые 5 минут проверяет состояние позиций
- Считает equity, P&L, APY
- *Если ❌ — данные о портфеле устаревают*

### 🔵 Мониторинг рисков

**`peg_monitor`** — Отслеживает привязку стейблкоинов
- Каждые 5 минут проверяет: USDC, DAI, USDT, sUSDS не отвязались ли от $1
- Если de-peg > порога — отправляет Red Flag в Telegram
- *Критично: при де-пеге нужно срочно выводить деньги*

**`red_flag_monitor`** — Общий монитор рисков
- Каждые 5 минут проверяет все источники рисков
- Взломы протоколов, аномалии ликвидности, странные транзакции

**`governance_watcher`** — Следит за DAO голосованиями
- Каждые 15 минут проверяет Snapshot и Tally
- Присылает алерт если в Aave, Compound, MakerDAO идёт важное голосование
- *Именно он прислал "Aave V3 — Экстренное голосование" что ты видел*

**`sky_monitor`** — Следит за Sky/MakerDAO протоколом
- Мониторит параметры sUSDS, GSM (Governance Security Module)
- Важно потому что у нас T1 адаптер на sUSDS

### 🔵 Здоровье системы

**`cycle_gap_monitor`** — Проверяет что торговые циклы не пропускаются
- Каждые 5 минут: был ли запуск cycle_runner последние 35 минут?
- Если нет — значит что-то сломалось

**`cycle_health`** — Детальная диагностика цикла
- Каждые 5 минут: нормальный ли P&L, нет ли аномалий в equity
- Выдаёт health_score от 0 до 100

**`uptime_monitor`** — Следит за всеми остальными агентами
- Каждые 5 минут проверяет: все ли агенты живы?
- Именно он даёт данные для `/agents` команды в боте

**`base_gas_monitor`** — Мониторит цену газа в сети Base
- Нужно для стратегий на Base chain (Aave Base, Morpho Base)

### 🟡 Инфраструктура

**`bot_commands`** — Сам Telegram бот
- Работает постоянно, слушает входящие сообщения
- Обрабатывает команды, отправляет уведомления

**`httpserver`** — Веб-дашборд
- Держит открытым сервер на порту 8080
- Дашборд с 6 вкладками доступен по http://localhost:8080

**`cloudflared`** — Публичный доступ к дашборду ❌
- Должен создавать HTTPS тоннель чтобы смотреть дашборд с телефона
- Пока НЕ настроен

**`fund-api`** — API для Family Fund ⏸
- Сервер для инвесторов Family Fund
- Запускается по расписанию когда нужен

### 🟡 Автоматизация

**`autopush`** — Автоматический пуш в GitHub
- Каждые 90 минут пушит изменения в репозиторий
- Бэкап кода и истории торгов

**`analytics_tier_c`** — Фоновая аналитика
- Запускается каждый день в 05:00
- Прогоняет 180 аналитических модулей Tier C (исследовательские)

**`daily-paper-report`** — Ежедневный отчёт
- Генерирует PDF и Telegram сообщение с итогами дня

### ⏸ Плановые агенты (запускаются редко)

**`checkpoint-7day`** ⏸ — 7-дневный чекпоинт
- Запускается раз в неделю
- Сохраняет полный снапшот состояния системы

**`weekly_backup`** ⏸ — Еженедельный бэкап
- Делает архив всех данных раз в неделю

## Почему у агентов ✅ / ❌ / ⏸?

- **✅** — Агент работал недавно (последний запуск менее X минут назад)
- **❌** — Агент не запускался слишком долго ИЛИ plist не загружен в launchd
- **⏸** — Плановый агент (запускается раз в день/неделю, сейчас просто ждёт)

**Нормальная картина:** большинство должны быть ✅, плановые — ⏸, cloudflared — ❌ (пока не настроен).
