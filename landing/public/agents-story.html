The agent fleet · launchd

An autonomous control plane for capital.

SPA is run by a fleet of ~40 deterministic agents — every one a launchd process on the production host, every one obeying explicit rules. They feed live market data, allocate, gate every move through risk, rebalance, and prove it. The runtime is LLM-forbidden by design: no model decides where money goes.

01The daily cycle

One deterministic loop, every day.

At 08:00 UTC a single canonical agent runs the whole book. It reads the market, asks the strategies for a target, runs that target through the risk gate, rebalances the virtual $100k portfolio, then writes verifiable evidence and serves it. Nothing is improvised — each stage is code with a fixed contract.

1

Adapters

27 read-only protocol adapters pull live APY and TVL from whitelisted DeFi venues. Read-only — they never move funds.

2

Strategies

The multi-strategy runner (S0–S77) and allocator turn the feed into target weights, capped for concentration.

3

RiskPolicy gate

A deterministic gate vets the target: TVL ≥ $5M, per-protocol caps, ≥5% cash. approved=false is final and cannot be overridden.

4

Rebalance

The $100k virtual book is rebalanced; trades, equity curve and NAV are recorded.

5

Evidence & API

A hash-chained audit trail makes every NAV verifiable; the API publishes it to the dashboards.

02Risk never sleeps

A standing guard, between cycles.

The daily loop runs once; risk runs continuously. A ring of monitors watches held protocols every few minutes — red flags, stablecoin pegs, governance proposals, threat-signal feeds. When something crosses a line, the threat reactor arms the kill switch intraday, independent of the schedule. The kill switch can flatten the book instantly, and it logs to the same audit trail.

03Self-healing

A fleet that repairs itself.

Autonomy means surviving failure without a human. Every five minutes a healer agent checks the whole fleet: if a launchd agent has died, it revives it; if the daily cycle was missed, it recovers it. And because the healer itself could fail, a watchdog guards the healer — restarting it and the core services if they ever go down. Two small deterministic agents keep everything else alive.

04Validation pipeline

Nothing ships unproven.

Before a strategy can carry real weight it runs a Tier-1 gauntlet. Deeper analytics tiers compute net-of-cost performance and deflated Sharpe; a digest reconciles NAV and assembles the validation gate; a governance layer enforces a default-deny policy table. Only what clears the gate is packaged — and the same audit trail signs the result.
